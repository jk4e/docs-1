**使用方法**

`wandb job [OPTIONS] COMMAND [ARGS]...`

**概要**

W&B jobs を管理および表示するためのコマンド


**オプション**

| **オプション** | **説明** |
| :--- | :--- |


**コマンド**

| **コマンド** | **説明** |
| :--- | :--- |
| create | wandb run なしで、ソースから job を作成します。 |
| describe | Launch job を記述します。 |
| list | プロジェクト 内の jobs をリスト表示します |
