**使用方法**

`wandb init [OPTIONS]`

**概要**

ディレクトリー を Weights & Biases で構成します。

**オプション**

| **オプション** | **説明** |
| :--- | :--- |
| `-p, --project` | 使用する project。 |
| `-e, --entity` | project のスコープとなる entity。 |
| `--reset` | 設定をリセットします。 |
| `-m, --mode` | "online"、"offline"、または "disabled" を指定できます。デフォルトは online です。 |
