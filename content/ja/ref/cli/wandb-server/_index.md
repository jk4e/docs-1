**使用法**

`wandb server [OPTIONS] COMMAND [ARGS]...`

**概要**

ローカルの W&B サーバーを操作するための コマンド です。

**オプション**

| **オプション** | **説明** |
| :--- | :--- |

**コマンド**

| **コマンド** | **説明** |
| :--- | :--- |
| start | ローカルの W&B サーバーを起動します。 |
| stop | ローカルの W&B サーバーを停止します。 |
