**Usage**

`wandb artifact [OPTIONS] COMMAND [ARGS]...`

**Summary**

Artifacts を操作するためのコマンド


**Options**

| **Option** | **Description** |
| :--- | :--- |


**Commands**

| **Command** | **Description** |
| :--- | :--- |
| cache | artifact キャッシュを操作するためのコマンド |
| get | wandb から artifact をダウンロード |
| ls | wandb の プロジェクト 内の全ての artifact をリスト表示 |
| put | wandb に artifact をアップロード |
