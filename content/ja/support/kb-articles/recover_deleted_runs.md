削除された Runs を復元するには、次の手順を実行します。

- Project Overview ページに移動します。
- 右上隅にある3つのドットをクリックします。
- **Undelete recently deleted runs** を選択します。

**注記**:
- 過去7日以内に削除された Runs のみ復元できます。
- Undelete がオプションにない場合は、W&B API を使用してログを手動でアップロードできます。
