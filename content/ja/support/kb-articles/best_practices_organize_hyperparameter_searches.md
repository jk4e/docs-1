`wandb.init(tags='your_tag')` で一意のタグを設定します。これにより、Project Page の Runs Table で対応するタグを選択して、プロジェクト の run を効率的にフィルタリングできます。

wandb.int の詳細については、[ドキュメント]({{< relref path="/ref/python/init.md" lang="ja" >}}) を参照してください。
